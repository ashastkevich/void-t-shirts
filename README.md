# VOID Store

Premium t-shirt store. Built with Next.js, Tailwind CSS v4, and motion/react.

## Stack

- **Framework:** Next.js 15 (App Router) — *migration in progress, currently Vite*
- **Styling:** Tailwind CSS v4
- **Animations:** motion/react
- **Database:** Supabase (PostgreSQL) + Prisma
- **Auth:** Supabase Auth
- **Deployment:** Vercel

## Development

```bash
npm install
npm run dev
```

## Environment variables

```
DATABASE_URL=
NEXT_PUBLIC_SUPABASE_URL=
NEXT_PUBLIC_SUPABASE_ANON_KEY=
```
